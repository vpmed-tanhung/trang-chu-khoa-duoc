CHỈ CẦN THAY 1 FILE

1. Giải nén file ZIP.
2. Trên GitHub, mở thư mục assets.
3. Tải đè file home_feature_sync.js trong thư mục assets.
4. Bấm Commit changes.
5. Chờ GitHub Pages cập nhật rồi nhấn Ctrl + F5 trên trang web.

KẾT QUẢ
- Các thẻ truy cập nhanh trong khung xanh được tạo tự động từ các thẻ tại mục “Ứng dụng chính”.
- Khi thêm một công cụ mới vào “Ứng dụng chính”, thẻ truy cập nhanh tương ứng tự xuất hiện.
- Khi đổi tên hoặc xóa công cụ, dãy thẻ cũng tự đồng bộ.
- Nhấn thẻ sẽ mở đúng công cụ tương ứng.
- “Nguồn dữ liệu hệ thống” không được đưa vào dãy thẻ vì đây là mục tham khảo, không phải công cụ chính.
