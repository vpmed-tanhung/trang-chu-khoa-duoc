HƯỚNG DẪN CHÈN MODULE VÀO WEB ĐANG CHẠY

1) Copy các file/thư mục này vào repository GitHub Pages:

- cap-nhat-du-lieu.html
- clinical-update-module.js
- data/clinical_sources.json
- data/drug_schema.json
- data/clinical_drug_template.json
- data/update_log.json

2) Mở file index.html đang dùng.

3) Tìm dòng cuối gần cuối file:

</body>

4) Chèn ngay phía trên nó:

<script src="clinical-update-module.js"></script>

5) Lưu file, commit, push.

6) Mở lại web:

https://vpmed-tanhung.github.io/trang-chu-khoa-duoc/#home

7) Trên trang chủ sẽ có thêm nút:

Cập nhật dữ liệu chuyên môn

8) Có thể mở trực tiếp trang quản trị cập nhật:

https://vpmed-tanhung.github.io/trang-chu-khoa-duoc/cap-nhat-du-lieu.html

LƯU Ý:
- GitHub Pages không tự ghi dữ liệu vào server.
- Dữ liệu nhập trong module sẽ lưu trên trình duyệt hiện tại.
- Sau khi nhập xong phải bấm “Xuất JSON”, rồi upload/commit file JSON vào thư mục data/.
- Nếu muốn cập nhật trực tiếp từ nhiều máy, cần kết nối Google Sheets + Apps Script hoặc backend riêng.
