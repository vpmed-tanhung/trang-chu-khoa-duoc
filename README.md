# Hệ thống Hỗ trợ Dược lâm sàng VPMED Tân Hưng - V4

Bản V4 cập nhật theo yêu cầu:
- Tích hợp toàn bộ 34 thuốc kháng sinh trong danh mục điều trị nội trú đã cung cấp.
- Không còn dữ liệu demo/ô “cần nhập” trong phần tra cứu kháng sinh.
- Bổ sung thẻ ứng dụng “Kiểm tra tương tác thuốc” với trạng thái “Đang phát triển”.
- Giữ liên kết sang công cụ tính liều kháng sinh & CrCl và phiếu đánh giá phần mềm.
- Dữ liệu hiển thị theo thẻ: tổng quan, liều & CrCl, HD/CRRT, pha truyền, tương tác ưu tiên, TDM/theo dõi, ADR.

## Cách cập nhật bằng GitHub Desktop
1. Giải nén file ZIP.
2. Xóa nội dung cũ trong thư mục local `Documents/GitHub/trang-chu-khoa-duoc`.
3. Copy toàn bộ nội dung bên trong thư mục V4 vào `Documents/GitHub/trang-chu-khoa-duoc`.
4. Mở GitHub Desktop → Summary: `Release V4 - cập nhật danh mục kháng sinh nội trú`.
5. Bấm `Commit to main` → `Push origin`.
6. Mở lại trang GitHub Pages và nhấn `Ctrl + F5`.

## Lưu ý chuyên môn
Công cụ hỗ trợ tra cứu, không thay thế đánh giá lâm sàng. Trước khi dùng như khuyến cáo chính thức của bệnh viện, nên được Khoa Dược/Hội đồng thuốc và điều trị rà soát, phê duyệt.
